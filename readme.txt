Show VAT number as part of billing address in the WooCommerce API

== Description ==

This plugin extends WooCommerce and Booster for WooCommerce. It adds a new key "vat_number" to billing address when requested by API call. Also it will auto fill the VAT value when creating a manual order.

== Installation ==

1. Upload `woo-vat-api` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Install WooCommerce and Booster for WooCommerce
